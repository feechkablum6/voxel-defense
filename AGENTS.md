# Codex Agent Rules

- User-facing explanations must be in Russian unless the user asks otherwise.
- Keep the project modular: no source file may exceed 700 lines of code.
- Do not put component or page-specific styles in global CSS. Keep styles near the owning UI code.
- Keep Phaser scenes thin. Gameplay rules belong in `src/game/simulation`.
- Do not add features outside the agreed MVP without asking first.
- Verify with `npm test` and `npm run build` before claiming the work is complete.
